# how to run
## run the server
* ```cd ./server```
* ```npm start```

API port is 8081 (e.g. http://localhost:8081/listUsers)

## run the client (second terminal)
* ```cd ./client```
* ```npm start```

Clients port is 3000 - http://localhost:3000/
